# 2.2 Robot Mobile App Connection (Agibot Go APP)

The latest app version must match the robot’s latest firmware version. Please download the correct version.

---

## 2.2.1 Download Links

### iOS

- Search the App Store for *Agibot Go APP*
- Scan the QR code below to download Agibot Go APP

### Android

### Operation Steps

#### Add Robot

1. Open the “Add Robot” page.

   [![打开添加界面](../_images/add_robot_1.png)](../_images/add_robot_1.png)

   Open the add page
2. Add the robot.

   [![添加机器人](../_images/add_robot_2.png)](../_images/add_robot_2.png)

   Add Robot

---

#### Connect to the robot via Wi-Fi

1. Tap **Wi-Fi Connection**.

   [![Wi-Fi 连接](../_images/wifi_1.png)](../_images/wifi_1.png)

   Wi-Fi Connection
2. Enter the Wi-Fi password.

   [![输入 Wi-Fi 密码](../_images/wifi_2.png)](../_images/wifi_2.png)

   Enter Wi-Fi password

---

#### Connect to the robot via AP Hotspot

1. Tap **AP Hotspot Connection**.

   [![AP 连接](../_images/ap_1.png)](../_images/ap_1.png)

   AP Connection
2. Select the connection.

   [![选择连接](../_images/ap_2.png)](../_images/ap_2.png)

   Select connection

---
